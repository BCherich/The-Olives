Stripe Webhook Endpoint

Add this in Stripe Dashboard → Developers → Webhooks to receive payment events.

🔑 How to get Stripe keys:
1. Create a free account at dashboard.stripe.com
2. Go to Developers → API Keys
3. Copy your Publishable Key (pk_...) and Secret Key (sk_...)
4. Start with test keys — switch to live keys when ready to accept real donations

Your M-Pesa Paybill number or Buy Goods (Till) number
Found in your Safaricom Daraja portal under the STK Push app credentials
M-Pesa Callback URL
https://your-domain.vercel.app/api/mpesa-callback

Register this URL in your Safaricom Daraja portal as the callback URL for STK Push.

📱 How to get M-Pesa Daraja keys:
1. Register at developer.safaricom.co.ke
2. Create an app → enable Lipa Na M-Pesa Online
3. Copy your Consumer Key and Consumer Secret
4. Find your Passkey in the STK Push app credentials
5. Use Sandbox for testing, switch to Production when ready to go live
